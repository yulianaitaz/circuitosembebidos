Detailed Description

Configuración central del sistema Bioreactor Fermentación Café.

Este archivo concentra todos los pines GPIO, constantes de límites físicos, tiempos del sistema, la estructura de datos de fermentación y las variables globales compartidas entre tareas FreeRTOS.

Enumeration Type Documentation

◆ estado_sistema_t

enum estado_sistema_t

Máquina de estados principal del bioreactor.

Flujo normal:

REPOSO → (botón inicio) → MONITOREO → (alerta detectada) →
ALERTA → (persiste > TIEMPO_MAX) → ALARMA
Enumerator
ESTADO_REPOSO 

Sistema encendido pero sin proceso activo. LED verde fijo.

ESTADO_MONITOREO 

Proceso activo, todos los parámetros normales. LED azul.

ESTADO_ALERTA 

Parámetro(s) fuera de rango. LED amarillo parpadeante.

ESTADO_ALARMA 

Alerta persistente o múltiple crítica. LED rojo parpadeante.
