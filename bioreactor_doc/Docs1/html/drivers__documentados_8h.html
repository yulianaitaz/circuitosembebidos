◆ dht_read_data()

esp_err_t dht_read_data ( int gpio_num,
float * humedad,
float * temperatura )

Lee temperatura y humedad del sensor DHT22.

Genera la señal de inicio, recibe 40 bits de datos y valida el checksum antes de convertir a valores físicos.

Parameters
[in]gpio_numNúmero del GPIO al que está conectado el sensor.
[out]humedadPuntero donde se almacenará la humedad relativa (%).
[out]temperaturaPuntero donde se almacenará la temperatura (°C).
Return values
ESP_OKLectura exitosa y checksum válido.
ESP_FAILError de comunicación (timeout) o checksum incorrecto.

◆ ds1302_leer_tiempo()

void ds1302_leer_tiempo ( rtc_tiempo_t * tiempo)

Lee la hora y fecha actuales del DS1302.

Si los datos recibidos son inválidos (mes > 12, día > 31, etc.), el driver simula el avance del tiempo sumando 5 segundos al último valor válido conocido.

◆ ds18b20_leer_temp()

esp_err_t ds18b20_leer_temp ( int gpio_num,
float * temperatura )

Lee la temperatura del sensor DS18B20.

Secuencia: Reset → Skip ROM (0xCC) → Convert T (0x44) → espera 800 ms → Reset → Skip ROM → Read Scratchpad (0xBE) → lee 2 bytes y convierte a grados Celsius.

Parameters
[in]gpio_numGPIO conectado al sensor.
[out]temperaturaPuntero donde se guarda la temperatura (°C).
Return values
ESP_OKTemperatura leída correctamente.
ESP_FAILSin respuesta del sensor en el bus 1-Wire.

void lcd_init ( void )

Inicializa el bus I2C y la pantalla LCD.

Configura el bus I2C en modo maestro, registra el dispositivo PCF8574 y ejecuta la secuencia de inicialización HD44780 en 4 bits: modo 2 líneas, cursor OFF, incremento automático, limpieza inicial.

◆ ph_init()

void ph_init ( int gpio_num)

Inicializa el ADC1 para la lectura del sensor de pH.

Configura la unidad ADC1, atenuación de 12 dB (rango 0–3.3 V) y resolución por defecto. El canal queda fijo en ADC_CHANNEL_0 (GPIO36).

◆ tarea_sensores()

void tarea_sensores ( void * pvParameters)

Tarea FreeRTOS de adquisición de datos del bioreactor.

Inicializa todos los sensores (DHT22, DS18B20, pH, LDR, RTC DS1302) y entra en un bucle de lectura periódica. Calcula el código de alerta combinado y publica un datos_fermentacion_t en xColaDatos.

Parameters
pvParametersNo utilizado (firma estándar FreeRTOS).
Note
Esta función es creada como tarea con xTaskCreate() en app_main().
